# 🚀 php002 - Sistema PHP com Login

Projeto desenvolvido para prática de PHP puro em ambiente LAMP, com foco em autenticação de usuários, sessões e organização de aplicações web.

---

## 📌 Objetivo

Este projeto tem como objetivo:

- Praticar PHP puro sem frameworks
- Entender autenticação com sessões
- Organizar melhor a estrutura do projeto
- Servir como projeto de portfólio

---

## ⚙️ Tecnologias utilizadas

- PHP
- MySQL / MariaDB
- HTML, CSS, JavaScript
- Apache (LAMP)

---

## 🔐 Funcionalidades atuais

- Login de usuário
- Controle de sessão
- Proteção de rotas (dashboard)
- Logout

---

## 📁 Estrutura do projeto

```bash
php002/
├── assets/      # Imagens e arquivos estáticos
├── config/      # Configurações (ex: conexão com banco)
├── css/         # Estilos
├── includes/    # Arquivos PHP reutilizáveis (header, footer, auth, etc)
├── js/          # Scripts JavaScript
└── *.php        # Arquivos principais (login, dashboard, etc)
```
